## Installation

- Clone the repository
```bash
$ git clone https://github.com/SpaciousCoder78/ghz-sh.git
```
- Move to project directory
```bash
$ cd ghz-sh
```
- Compile the `main.c` file
```bash
$ gcc -o ghz-sh main.c
```
- Run the executable file
```bash
$ ./ghz-sh
```
